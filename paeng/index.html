Mango Pest Identification

Project

LIn this project we developed software for identifying pests commonly found on mango trees. The software detects the mango pests from input photographs and counts the number of them found in the photographs. The software employed both the image processing and machine learning techniques identify the insects. The detection model was trained on a typical personal computer and was later deployed to run on a mobile phone. For this project we focus on the commonly found pest on mango trees; fruit flies, leafhoppers, weevils, and leaf beetles. 3000 images of various kinds of insects were used in the training. The results were summarized as follows: accuracy 82.5 percents, precision 83 percents, sensitivity 81 percents, and specificity 92.6 percents.
